# Herramientas y Utilidades

El portal cuenta con una serie de páginas especializadas para tareas institucionales específicas.

- **Gestión Técnica**: `cita.html` (Manuales técnicos administrables).
- **Ambientales**: `respel.html`, `rua.html`, `pcb.html`.
- **Soporte**: `correo.html`, `soporte.html`.
- **Consulta**: `busqueda.html`, `calendario.html`, `galeria.html`, `cartografia.html`.

Todas estas páginas utilizan el sistema de diseño "Premium" y navegación unificada.
